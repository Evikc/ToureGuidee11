# ToureGuidee - Android приложение для уведомления о достопримечательностях

## Описание
ToureGuidee - это Android-приложение, которое уведомляет пользователя о ближайших достопримечательностях в радиусе 500 метров. Приложение работает в реальном времени с использованием GPS и отображает достопримечательности на карте.

## Основные функции

1. **Карта достопримечательностей** - Отображение всех добавленных достопримечательностей на интерактивной Google Map
2. **GPS-отслеживание** - Мониторинг местоположения пользователя в реальном времени
3. **Уведомления** - Автоматические push-уведомления при приближении к достопримечательности (радиус 500м)
4. **Добавление достопримечательностей** - Пользователь может добавлять свои собственные достопримечательности
5. **Карточка достопримечательности** - Подробная информация с кнопкой "Проложить маршрут"
6. **Список достопримечательностей** - Просмотр всех добавленных объектов в виде списка

## Настройка

### 1. Получение Google Maps API Key
1. Перейдите в [Google Cloud Console](https://console.cloud.google.com/)
2. Создайте новый проект или выберите существующий
3. Включите **Maps SDK for Android**
4. Создайте credentials (API Key)
5. Добавьте SHA-1 fingerprint вашего debug/release ключа

### 2. Добавление API Key
Откройте `app/src/main/res/values/strings.xml` и замените `YOUR_API_KEY_HERE` на ваш API ключ:
```xml
<string name="map_api_key">ВАШ_API_KEY</string>
```

Или создайте файл `local.properties` в корне проекта:
```
MAPS_API_KEY=ВАШ_API_KEY
```

### 3. Разрешения
Приложение запрашивает следующие разрешения:
- `ACCESS_FINE_LOCATION` - точное местоположение
- `ACCESS_COARSE_LOCATION` - приблизительное местоположение
- `POST_NOTIFICATIONS` - отправка уведомлений (Android 13+)
- `ACCESS_BACKGROUND_LOCATION` - работа в фоновом режиме

## Сценарий использования

1. **Запуск приложения** → Пользователь видит карту с достопримечательностями
2. **Добавление достопримечательности**:
   - Нажмите "+" на экране списка
   - Введите название, описание и координаты
   - Сохраните
3. **Получение уведомления**:
   - Приближайтесь к достопримечательности
   - При расстоянии ≤ 500м приходит уведомление
4. **Просмотр информации**:
   - Нажмите на уведомление или маркер на карте
   - Откроется карточка с описанием
   - Нажмите "Проложить маршрут" для навигации

## Сборка и запуск

```bash
# Клонирование репозитория
git clone <repository-url>
cd ToureGuidee

# Сборка debug версии
./gradlew assembleDebug

# Установка на устройство
./gradlew installDebug

# Или откройте проект в Android Studio и нажмите Run
```
